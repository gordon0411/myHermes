# 亚马逊选品分析6步法

## 核心铁律（每次选品前必读）

1. **所有数据指标必须标注来源** — 每个数字都要说明"从哪来的"
   - ✅ `"CPC $0.32（来源：卖家精灵MCP keyword_research，2026-03）"`
   - ❌ `"CPC大概$0.2-0.3"` — 没有来源，禁止

2. **CPC必须从卖家精灵MCP获取** — 用`traffic_keyword`/`keyword_research`的`bid`字段，禁止从广告密度推断
3. **绝不拍脑袋给评分** — 所有关键指标必须有实际数据支撑
4. **先看品类全貌，再看单个产品** — 单个产品数据再漂亮，放到品类里可能毫无竞争力
5. **评论壁垒是选品第一指标** — 头部评论>10K的品类，新品几乎不可能突围

---

## 选品分析6步法

### 步骤1: 标杆产品数据采集

**操作**: 用browser tool访问产品页面，采集字段：
- ASIN、品牌、卖家类型（1P/3P）
- 售价、变体数量及价格范围
- 评分、评论数、月销标签
- BSR（Best Sellers Rank）— 大类+子类
- 上架时间、材质、尺寸、重量
- Amazon's Choice / Best Seller 标签

---

### 步骤2: 关键词矩阵构建（最关键）

**三步挖词法**：

**第①步：反查标杆ASIN流量词**
```bash
mcporter call sellersprite-mcp.traffic_keyword \
  --args '{"request":{"asin":"标杆ASIN","marketplace":"US","size":20,"order":{"field":"searches","desc":true}}}' \
  --output json
```

**第②步：大词向下拆解**
```bash
mcporter call sellersprite-mcp.keyword_research \
  --args '{"request":{"keywords":"大词","marketplace":"US","size":20}}' \
  --output json
```

**第③步：补充拓展词**
```bash
mcporter call sellersprite-mcp.traffic_extend \
  --args '{"request":{"asinList":["标杆ASIN","竞品ASIN1"],"marketplace":"US","queryType":2,"size":20}}' \
  --output json
```

**关键词分层标准**：
| 层级 | 月搜索量 |
|------|----------|
| 大词 | > 50,000 |
| 核心词 | 10,000 - 50,000 |
| 细分词 | 1,000 - 10,000 |
| 长尾词 | < 1,000 |

---

### 步骤3: 市场集中度计算

| 指标 | 计算方法 | 红线 |
|------|----------|------|
| CR1（品牌） | 最大品牌月销 / 品类总月销 | >35% → 🔴 |
| CR3（品牌） | 前三品牌月销合计 / 品类总月销 | >60% → 🔴 |
| Top1 ASIN占比 | 头部单链接月销 / 品类总月销 | >20% → 🔴 |

**判断标准**：
- CR3 < 40% → 🟢 分散市场，新品有机会
- CR3 40-55% → 🟡 中等集中，需差异化
- CR3 > 55% → 🔴 高度集中，慎入

---

### 步骤4: 流量集中度分析

**关键判断**：
- 大词搜索量 vs 细分词搜索量差距 >20倍 → 流量高度集中在大词
- 细分词搜索量 < 5,000 → 天花板低
- 大词+核心词+细分词合计搜索量 < 10,000 → 品类偏小众

---

### 步骤5: 新品广告盈利测算

**ACOS公式**：
```
ACOS = CPC / (转化率 × 售价)
```

**转化率参考**：
| 阶段 | 转化率 |
|------|--------|
| 冷启动（0评论） | 5-8% |
| 有评论后（50+） | 10-15% |
| 成熟期（200+） | 15-20% |

**判断标准**：
- 冷启动ACOS < 40% → 🟢 广告可盈利
- 冷启动ACOS 40-60% → 🟡 需要烧钱期，有风险
- 冷启动ACOS > 60% → 🔴 广告很难盈利

**⚠️ 补充：毛利测算（步骤5扩展）**
```
毛利 = 售价 - 采购成本 - 头程 - FBA履约费 - 广告费
     = 售价 × (1 - 平台佣金15%) - 采购 - 头程 - FBA - 广告
```
- 毛利 > 30% → 🟢 推荐
- 毛利 20-30% → 🟡 可接受
- 毛利 < 20% → 🔴 慎入

---

### 步骤6: 综合评判（6维度评分）

| 维度 | 5分 | 3分 | 1分 |
|------|-----|-----|-----|
| 市场集中度 | CR3<40% | CR3 40-55% | CR3>60% |
| 流量可获取性 | 细分词月搜>20K且广告密度<20% | 细分词月搜5K-20K | 大词被锁死 |
| CPC可行性 | 冷启ACOS<40% | 冷启ACOS 40-60% | 冷启ACOS>60% |
| 评论壁垒 | 头部<1K评论 | 头部1K-10K | 头部>10K |
| 差异化空间 | 有明确痛点可改进 | 一般差异化 | 纯白牌同质化 |
| 利润空间 | 毛利>30% | 毛利20-30% | 毛利<20% |

**决策标准**：
- ≥ 24分 → 🟢 推荐
- 18-23分 → 🟡 谨慎
- < 18分 → 🔴 放弃

**⚠️ 一票否决条件（直接放弃）**：
1. 头部评论 > 20K 且 CR1 > 30%
2. 冷启动ACOS > 80%
3. 品类Top1 BSR在大类前100名
4. 单一品牌占搜索前48名中 > 6个坑位

## 工具与数据源

| 优先级 | 工具 | 用途 |
|--------|------|------|
| 优先级1 | 卖家精灵MCP | CPC/搜索量/集中度的唯一可靠来源 |
| 优先级2 | Amazon搜索结果 | 竞争格局验证 |
| 优先级3 | 1688 | 供应链验证 |

**⚠️ CPC数据铁律**：
- 绝对禁止从广告密度反推CPC（实测偏差达2.5-4倍）
- 必须用卖家精灵MCP的`bid`字段

---

## 常见坑

| 坑 | 表现 | 正确做法 |
|----|------|----------|
| CPC推断失真 | 广告密度推CPC偏差2.5-4倍 | 必须用卖家精灵MCP的bid字段 |
| 忽视品牌矩阵 | 只看单ASIN，没发现同品牌8个ASIN | 搜索结果统计品牌占位 |
| 细分词幻觉 | 细分词看起来竞争低，但流量池太小 | 看细分词头部月销天花板 |

## 扩展：趋势/季节性判断

在步骤4之后、步骤5之前，增加**趋势判断**：

1. **Google Trends验证**：搜索品类大词近3年趋势
   - 上升期 → 🟢 顺势而为
   - 平稳期 → 🟡 差异化竞争
   - 下降期 → 🔴 慎入

2. **季节性判断**：检查是否有明显淡旺季
   - 节日类（万圣节、圣诞）→ 需提前3-6个月备货
   - 全年平稳 → 稳定运营

3. **新品推广期估算**：
   - 达到50评论所需时间（月）
   - 冷启动期广告预算估算
   - 回本周期计算

## 扩展：竞品ASIN调研方法

### 获取竞品数据来源

1. **搜索入口**：
   - Amazon Best Sellers页面（类目节点）
   - 关键词搜索结果前3页
   - `site:amazon.com "关键词" best seller`

2. **市场数据工具**：
   - Asinsight（asin级别月销、关键词数据）
   - AMZChart（竞争度评分）
   - Jungle Scout / Helium10

3. **提取竞品核心字段**：
   ```python
   # 从Amazon页面提取的关键字段
   {
       "asin": "B079CXWYYP",
       "brand": "Lusso Gear",
       "price": 15.99,
       "rating": 3.8,
       "reviews": 18924,
       "bsr": "#5,168 Automotive",
       "first_available": "2023-08",
       "material": "Vegan Leather",
       "negative_reviews": ["moves around", "slides down", "fit issues"]
   }
   ```

### 竞品痛点分析方法

从评论中提取负面反馈，识别差异化机会：

| 痛点模式 | 含义 | 差异化方向 |
|----------|------|------------|
| "moves around" | 稳定性差 | 防滑底部设计 |
| "slides down" | 下沉滑动 | 硬底支撑+防滑 |
| "doesn't fit" | 适配性差 | 可调节楔形块 |
| "poor quality" | 材质差 | 升级材质 |
| "too narrow/wide" | 尺寸问题 | 多种尺寸规格 |

### 竞品对标表模板

| 竞品 | ASIN | 评论数 | 月销 | 价格 | 评分 | BSR | 痛点 | 启示 |
|------|------|--------|------|------|------|-----|------|------|
| Drop Stop | B07XXX | 73,595 | 10,000+ | $24.99 | 4.6 | #100 | 先发优势 | 避开 |
| Lusso Gear | B079CXWYYP | 18,924 | 900+ | $15.99 | 3.8 | #5K | 稳定性差 | 差异化机会 |
| Owntow | B0D4QK9FVQ | 353 | 300+ | $39.90 | 4.4 | #5,168 | 新品 | 证明高品质可行 |

### 差异化策略框架

基于竞品痛点，制定差异化方案：

```
痛点 → 设计解决方案 → 成本增加 → 定价策略
```

**示例**：
- 竞品痛点："moves around"（稳定性差）
- 设计方案：底部加防滑硅胶颗粒
- 成本增加：$0.3-0.5/件
- 定价策略：$17.99 vs 对手$15.99（溢价$2=12.5%）

### 切入价格区间判断

| 价格区间 | 定位 | 策略 |
|----------|------|------|
| $10-$14 | 性价比 | 低价走量，利润薄 |
| $15-$20 | 主流竞争 | 品质差异化主战场 |
| $20-$25 | 高端 | 品质+品牌支撑 |
| >$25 | 奢侈 | 需要强品牌背书 |
