# Hermes Agent 使用说明

## 安装状态
✅ Hermes Agent 已成功安装并配置完成

## 快速开始

### 方法一：使用启动脚本（推荐）
在 PowerShell 中运行：
```powershell
.\start-hermes.ps1
```

### 方法二：手动设置环境变量
在 PowerShell 中运行：
```powershell
$env:HERMES_HOME = "d:\GuojinX\xilu"
hermes
```

## 飞书集成

### 启动飞书网关
由于Windows兼容性问题，请使用以下命令：
```powershell
cd d:\GuojinX\xilu
python start-hermes-gateway-v2.py
```

### Web Dashboard（控制台）
启动命令：
```powershell
hermes dashboard
```
地址：**http://127.0.0.1:9119**

功能：
- 查看连接状态
- 管理会话历史
- 配置管理

## 常用命令

| 命令 | 说明 |
|------|------|
| `hermes` | 启动交互式聊天 |
| `hermes chat -q "问题"` | 单次提问模式 |
| `hermes --resume <session_id>` | 恢复之前的会话 |
| `hermes model` | 选择默认模型 |
| `hermes config` | 查看配置 |
| `hermes doctor` | 检查配置和依赖 |
| `hermes logs` | 查看日志 |
| `hermes gateway` | 启动消息网关 |
| `hermes dashboard` | 启动Web控制台 |
| `hermes pairing approve feishu <code>` | 批准飞书用户配对 |

## 飞书命令

## 文件说明

- `.env` - 环境变量配置文件（包含API密钥和飞书配置）
- `start-hermes.ps1` - 标准启动脚本
- `start-hermes-gateway-v2.py` - 飞书网关启动脚本（Windows兼容）
- `使用说明.md` - 本文档
- `logs/` - 日志目录
- `sessions/` - 会话历史
- `memories/` - 记忆存储

## 注意事项

1. 始终在 `d:\GuojinX\xilu` 目录下运行

3. 如需更换 API key，请编辑 `.env` 文件
4. Windows上启动飞书网关请使用 `start-hermes-gateway-v2.py`
