# Hermes 飞书网关后台服务安装说明

## 方法一：以管理员身份运行（推荐）

1. 右键点击 PowerShell 图标
2. 选择 **"以管理员身份运行"**
3. 运行以下命令：

```powershell
cd d:\GuojinX\xilu
.\install-hermes-service.ps1
```

## 方法二：手动创建开机自启

1. 右键点击 PowerShell 图标，选择 **"以管理员身份运行"**
2. 运行以下命令：

```powershell
$action = New-ScheduledTaskAction -Execute "C:\Users\admin.ZBYCORP\AppData\Local\hermes\hermes-agent\venv\Scripts\python.exe" -Argument "`"d:\GuojinX\xilu\start-hermes-gateway-v2.py`"" -WorkingDirectory "d:\GuojinX\xilu"
$trigger = New-ScheduledTaskTrigger -AtLogOn
$settings = New-ScheduledTaskSettingsSet -AllowStartIfOnBatteries -DontStopIfGoingOnBatteries -StartWhenAvailable
Register-ScheduledTask -TaskName "HermesGateway" -Action $action -Trigger $trigger -Settings $settings -Description "Hermes Agent Gateway for Feishu"
```

## 常用命令

查看服务状态：
```powershell
Get-ScheduledTask -TaskName "HermesGateway" | Get-ScheduledTaskInfo
```

启动服务：
```powershell
Start-ScheduledTask -TaskName "HermesGateway"
```

停止服务：
```powershell
Stop-ScheduledTask -TaskName "HermesGateway"
```

卸载服务：
```powershell
Unregister-ScheduledTask -TaskName "HermesGateway" -Confirm:$false
```

Get-Content "d:\GuojinX\xilu\logs\gateway.log" -Wait -Tail 50
