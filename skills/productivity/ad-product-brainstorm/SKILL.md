# 新一代智能广告方向碰撞-会议

## 基本信息
- **时间**：明天（第一次开会）
- **参与方**：黄国进（数字广告侧）、子不语产品侧、广告客户交付侧
- **目标**：想法碰撞，不追求一次出结论

## 议程

### 第一部分：三方输入（20~30分钟，每人≤10分钟）
| 发言人 | 负责内容 |
|--------|---------|
| 黄国进 | 数字广告整体情况、想法碰撞 |
| 子不语侧同事 | 广告产品前期上线情况、功能、后续想法 |
| 客户交付侧同事 | 客户视角的输入 |
| 鬼才（AI） | 配合黄国进提供数字广告方面的输入 |

### 第二部分：方向定义
- 各方输入汇总，碰撞出新一代智能广告大致方向
- 初步定义"新一代智能广告亟待解决的问题"
- 形成下一步工作方向（产品设计、子不语迭代方向、外部在运行智能广告的待解决问题）

## 鬼才的任务
明天配合黄国进，准备**数字广告整体情况**的输入，内容包括：
1. 数字广告当前整体格局（平台侧、甲方侧、代理商侧）
2. 主流广告产品的核心逻辑（投放、创意、归因、优化）
3. 行业正在往哪个方向演进（AI搅动的地方在哪）
4. 我们自己的数字广告产品在行业里大概什么位置
